## 3、制作镜像

### 3.1 编写Dockerfile文件

```xml
FROM openjdk:8-jdk-alpine
VOLUME /tmp
ADD ./service_oss-0.0.1-SNAPSHOT.jar service_oss.jar
ENTRYPOINT ["java","-jar","/service_oss.jar", "&"]
```
### 3.2 上传jar包和Dockerfile文件

![image](images/83c8f70f-31f2-4db9-8e17-b536e7856d98.png)



### 3.3 进入ockerfile所在目录

**执行命令制作镜像：**

**docker build -t service-oss:1.0.0 .**

![image](images/0deaa17b-491e-4273-9300-fa881b9f02fe.png)

**容器启动测试：**

注：项目需要nacos环境，需要启动nacos服务，可以在Liunx中安装启动

**docker run -d -p 8205:8205 service-oss:1.0.0 -t**

![image](images/653d0a6b-defa-4b3b-937b-5eb8b39a2cd0.png)

**访问测试：**

![image](images/a42647dc-fc9c-437d-a852-2b3c5aca57c9.png)
